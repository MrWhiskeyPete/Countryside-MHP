# Countryside MHP Website

Static website for Countryside Mobile Home & RV Park — a 55+ community in Donna, TX (Rio Grande Valley).

**Live site:** [countrysidetexas.com](https://www.countrysidetexas.com)

## Structure
```
index.html       — Home page
about.html       — About the community
sites.html       — Sites & rentals (lots, RV pads, park-owned rentals)
amenities.html   — Amenities & activities
contact.html     — Contact form & info
css/style.css    — All styles (warm southwestern palette)
js/main.js       — Nav toggle & form handling
```

## Hosting
Hosted on Cloudflare Pages, deployed from this repo.

## Contact Form
The contact form uses [Formspree](https://formspree.io). Update the form action URL in `contact.html` with your Formspree endpoint.

## Making Changes
Edit the HTML/CSS files and push to `main`. Cloudflare Pages auto-deploys.
